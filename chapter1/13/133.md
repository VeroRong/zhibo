###1.3.3 编译安装nginx
**下载和解压**
    # cd /usr/local/
    # wget http://nginx.org/download/nginx-1.12.2.tar.gz
    # tar -zxvf nginx-1.12.2.tar.gz
**编译**
    # cd nginx-1.12.2
    # ./configure --help
“./configure --help”指令能列出大部分常用模块和编译选项，列出的编译选项中以--without开头的都默认安装，以PATH结尾的需要手动指定依赖库源码目录。
