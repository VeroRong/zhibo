###1.3.4 配置防火墙
nginx装好之后我们需要配置防火墙开启80端口，如果不开启，有时防火墙会阻止外网访问80端口，我们就无法访问nginx的配置网站。
**配置**
**启动nginx服务**
```
# /usr/local/nginx/sbin/nginx
```
**查看进程**
```
# ps -ef | grep nginx
```
nginx启动之后我们便可以查看进程。
