###1.4.3 启动、关闭和重启
####1.启动
方式一：
```
# /usr/local/nginx/sbin/nginx
```
方式二：
```
# service nginx start
```
####2.关闭
nginx服务停止的方法有三种：从容停止、快速停止和强制停止。一般都是通过发送系统信号给主进程的方式来停止nginx服务。
**（1）查看nginx主进程号**
```
# ps -ef | grep nginx
```
该指令我们已经在1.3.4小节中介绍过，这里便不再赘述。
**（2）从容停止**
```
# kill -QUIT 主进程号
```
**（3）快速停止**
**（4）强制停止**
```
# pkill -9 nginx
```
快速停止：# kill -TERM 主进程号
