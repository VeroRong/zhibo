###2.4 NRM的搭建
>首先请先参照第一章在linux中成功安装nginx
本文所使用的为Centos
使用工具Xshell

1.下载 nginx-rtmp-module


```
 打开 http://arut.github.io/nginx-rtmp-module/ 选择Download.tar.gz
 本文所下载使用的版本为arut-nginx-rtmp-module-v1.2.1-0-g791b613.tar.gz
```

2.安装 rz/sz命令

```
 yum install -y lrzsz
```
  
3.创建一个目录

```
mkdir /download
```
